    Summary

    This lab introduced you to developing applications for the Windows Phone platform using the XNA Framework. From this lab you created an XNA Game Studio game for Windows Phone, loaded the game’s resources, took care of the input, updated the game state, and added game specific logic.

    By completing this hands-on lab, you also became familiar with the tools required to create and test a Windows Phone XNA Game Studio game. In this lab, you created a new XNA Game Studio game for Windows Phone application using Microsoft Visual Phone Developer 2010 Express, and then used this free tool to create the application logic and the layout of the user interface.
