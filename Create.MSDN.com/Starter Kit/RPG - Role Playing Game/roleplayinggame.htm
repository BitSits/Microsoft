Role-Playing Game
Starter Kit
area
2D Graphics, Artificial Intelligence, Collision, Content Pipeline, Gameplay
submitted
5/19/2008

Description: 

The Role-Playing Game starter kit is a complete XNA Game Studio game, originally created for Microsoft by Dhruva Interactive. The game comes ready to compile and run, and you are free to use its source code as the basis for your own XNA Game Studio game projects, and to share your work with others.

This starter kit contains a two-dimensional role-playing game that utilizes a tile engine. This game is complete with character classes, multiple party members, items, and quests. It also features a turn-based combat engine with a side-on presentation with animated characters and effects. Additionally, all of the game data is exposed in XML files, making it easy to write your own quests, create your own towns or even write your own storyline.

 

A World of Possibilities

 

Modify Content
Modify existing content or create new content of your own. All gameplay content is loaded into the Content Pipeline via XML, making it easy to add, replace or modify assets you would like to include in your own Role-Playing Game.

Expand Combat Functionality
The Role-Playing Game starter kit comes complete with a dynamic combat system that can be quickly modified to produce exciting new results. You can create unique combat actions, expand the party size and even build a menu that allows you to switch characters in and out of combat.

Customize AI
Creating custom AI behaviors has never been easier. Do you want to expand the the list of available attacks and spells opponents have in combat? With a few simple modifications to the existing system, you can add in new features like enemy healing spells or status damaging attacks.

Reuse Systems in Your Own Games
Many of the engines and systems in the Role-Playing Game starter kit can be removed from the main game and reused in new projects including new games. 

To install the Role-Playing Game Starter Kit:

1. Download the Role-Playing Game .vsi files.

2. Run Visual Studio Installer, following the on-screen instructions.

3. On the Visual C# Express File menu, click New Project. This may take a minute to load the template for the first time.

4. Choose Role-Playing Game Starter Kit, and then click OK.

 

Tutorials and Documents

The Role-Playing Game starter kit includes a "Readme" file outlining the game controls, architecture, and ideas for improving and expanding game features. To access this document, simply follow the instructions above to install the starter kit - the document will display automatically.
