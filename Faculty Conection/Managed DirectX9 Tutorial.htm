Overview

Managed DirectX is a set of classes for creating games and other multimedia applications. It includes support for high-performance 2D and 3D graphics, sound and music, input, force feedback, multimedia streaming, and network communication.

Managed DirectX is quite new regarding the standard distribution, and not yet fully stable. So the documentation is still preliminary and subject to change.

The components provided by DirectX 9.0 are:

  • Direct3D that enable fast and easy rendering of 3D scenes
  • DirectDraw that provides a direct low-level access to video memory
  • DirectInput to control your application from any kind of input device
  • DirectPlay that make easier network communication between computers
  • DirectSound provides support for playing and capturing sound samples
  • Audio Video Playback allows control and playback of audio and video media

This is an introduction to Managed DirectX, mostly oriented to Direct3D which is the most complex component. When using only 2D drawing, it's worth to note that DirectDraw has no support for advanced features such as hardware rotation, alpha blending. If you need 2D basic capabilities, you could use DirectDraw that is probably a bit simpler than Direct3D.

First of all, we will see how to create a simple program that uses DirectX. Then all workshops will be run under a single demonstration application. Last, this program will be shelled to understand how it works.
