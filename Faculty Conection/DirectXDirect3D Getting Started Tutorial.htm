Overview

This tutorial from the 2004 Imagine Cup is designed as a beginner’s guide to getting started with DirectX 9.0 (specifically with Direct3D). It assumes no prior knowledge of computer graphics or of DirectX, but it does assume that the learner has experience using both C++ and Visual Studio. DirectX can be fairly intimidating to those who have no experience with computer graphics. This tutorial not only covers the technical details, but enables the learner to learn a little about the underlying computer graphics principles as well. While the end result of this tutorial is a relatively simple application, it should be enough to get the student started with DirectX. In addition to reading about how DirectX works, it is also equally useful to learn about the theory behind computer graphics (such as how the various matrices work for say rotation) as it will make it that much easier to pick up new ideas and concepts in DirectX if you have the background in computer graphics.

Tutorial Details

Tutorial: Getting Started

The first thing to do is to install Microsoft Visual Studio.
The second thing to do is install the DirectX 9.0 SDK.

It is best to install Visual Studio first and then to install the DirectX 9.0 SDK. Doing so will allow the SDK install to modify library paths in Visual Studio so that you won’t have to modify them individually for every project when linking to the DirectX libraries.

Another thing to note is that installing in this manner will create a wizard under “new project” which will allow the quick creation of a DirectX 9.0 application. This is great to play around with if you’re just starting with DirectX. However for this tutorial, we’ll be doing everything from scratch.
