Overview

The programs are very simple and consistent in nature to remove any distracting and unnecessary source code. They are not based on the scene graph previously described. The simple examples are structured this way in order to allow the students to experience integrating HLSL shader code into a fixed function scene graph, one of the key learning outcomes for the unit in which this code is used. To use these demos, simply unpack the Shader Demo *.zip files, open the directory of the program you wish to use, open the Visual Studio project file, and compile and run the program. Each demo assumes that you have a valid installation of DirectX 9.0c.
